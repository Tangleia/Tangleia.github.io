案例

用户输入用户名，密码以及验证码。

  • 如果用户名和密码输入有误，跳转登录页面，提示:用户名或密码错误
  • 如果验证码输入有误，跳转登录页面，提示：验证码错误
  • 如果全部输入正确，则跳转到主页success.jsp，显示：用户名,欢迎您

思路

在生成验证码的时候将它的值存入到session中，在比对的时候再取出来进行对比
